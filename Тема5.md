# Тема 5  
# Типи та характеристики SSD-накопичувачів

## Вступ

У сучасних персональних комп’ютерах усе частіше використовуються SSD-накопичувачі замість традиційних жорстких дисків (HDD). Це пояснюється їх високою швидкістю роботи, надійністю та енергоефективністю.  

Метою даної роботи є розгляд основних типів SSD, їх технічних характеристик та створення скрипта для перегляду інформації про встановлений накопичувач у Windows.

---

## 1. Що таке SSD

SSD (Solid State Drive) — це твердотільний накопичувач, який використовує флеш-пам’ять для зберігання даних.  

На відміну від HDD, SSD:
- не має рухомих механічних частин;
- працює значно швидше;
- є безшумним;
- споживає менше електроенергії;
- стійкий до механічних пошкоджень.

---

## 2. Основні типи SSD

### 2.1 SATA SSD

SATA SSD підключається через інтерфейс SATA III (до 6 Гбіт/с).  
Це найпоширеніший варіант для модернізації старих ПК.

**Характеристики:**
- Швидкість читання: до 550 МБ/с  
- Форм-фактор: 2.5"  
- Інтерфейс: SATA III  

**Переваги:**
- доступна ціна;
- сумісність із більшістю комп’ютерів.

**Недоліки:**
- обмежена швидкість через інтерфейс SATA.

---

### 2.2 M.2 SATA SSD

M.2 SATA SSD — це компактна версія SATA SSD у форм-факторі M.2.  
Часто використовується в ноутбуках.

**Характеристики:**
- Форм-фактор: M.2  
- Інтерфейс: SATA  
- Швидкість: до 550 МБ/с  

---

### 2.3 M.2 NVMe SSD

NVMe SSD працює через інтерфейс PCI Express і використовує сучасний протокол NVMe.

**Характеристики:**
- Інтерфейс: PCIe  
- Протокол: NVMe  
- Швидкість читання: 2000–7000 МБ/с  
- Форм-фактор: M.2  

**Переваги:**
- дуже висока швидкість;
- мінімальна затримка;
- підходить для ігор і професійних програм.

**Недоліки:**
- вища ціна;
- потребує підтримки материнської плати.

---

### 2.4 PCIe SSD (плата розширення)

Це SSD, який встановлюється у слот PCIe як окрема плата розширення.  
Використовується переважно у серверах та високопродуктивних робочих станціях.

---

## 3. Основні характеристики SSD

При виборі SSD слід звертати увагу на такі параметри:

### 3.1 Об’єм пам’яті

Вимірюється у GB або TB.  
Найпоширеніші варіанти: 256 ГБ, 512 ГБ, 1 ТБ.

### 3.2 Швидкість читання та запису

Визначає, наскільки швидко накопичувач працює з файлами.  
NVMe значно швидший за SATA.

### 3.3 Тип флеш-пам’яті

| Тип | Особливість |
|------|------------|
| SLC | Найнадійніший, дорогий |
| MLC | Середній ресурс |
| TLC | Найпоширеніший варіант |
| QLC | Нижчий ресурс, дешевший |

### 3.4 TBW (Total Bytes Written)

Показує загальний обсяг даних, який можна записати на диск протягом його ресурсу.

### 3.5 IOPS

Кількість операцій введення-виведення за секунду.  
Важливо для серверних систем та баз даних.

### 3.6 Інтерфейс підключення

- SATA  
- PCIe  
- NVMe  

### 3.7 SMART-показники

Дозволяють оцінити стан здоров’я накопичувача та рівень його зношення.

---

## 4. Отримання технічної інформації у Windows

Для перегляду детальної інформації про встановлений накопичувач було створено окремий BAT-файл (CMD), який відображає:

- модель диска;
- виробника;
- серійний номер;
- тип інтерфейсу (SATA, NVMe, USB);
- тип носія (SSD або HDD);
- об’єм;
- файлову систему;
- вільний простір.

Скрипт додається до репозиторію окремим файлом.

---

## Висновок

У ході виконання роботи було розглянуто основні типи SSD-накопичувачів та їх характеристики. SSD значно перевершують HDD за швидкістю, надійністю та енергоефективністю.  

Найбільш продуктивним варіантом для сучасних комп’ютерів є NVMe SSD, оскільки він забезпечує максимальну швидкість передачі даних.  

Також було створено окремий командний файл для отримання технічної інформації про встановлений накопичувач у середовищі Windows.
