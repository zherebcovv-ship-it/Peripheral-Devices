# Тема 4. Слоти розширення та карти

## 1. Що таке слоти розширення на материнці

Слоти — це довгі роз'єми на платі, куди вставляють додаткові карти. Зараз головний — PCIe (PCI Express). Вони бувають різні за розміром:

- PCIe x1 — маленький, короткий. Для Wi-Fi, звукової карти, простих штук.
- PCIe x4 — середній. Для SSD (M.2 через перехідник), карт захоплення.
- PCIe x8 — довший. Для швидких мереж або контролерів.
- PCIe x16 — найдовший. Для відеокарти, майже завжди.

Чим більше "x" (ліній) — тим швидше працює. Маленьку карту можна ставити в великий слот, але не навпаки.

## 2. 8 прикладів карт PCIe (які вставляють)

1. Відеокарта — в x16. Головна для ігор (RTX, Radeon).
2. Wi-Fi карта — в x1. Додає бездротовий інтернет + Bluetooth.
3. Звукова карта — в x1. Кращий звук, ніж вбудований.
4. Картка захоплення відео — в x4. Щоб записувати ігри або ТБ.
5. Адаптер для SSD (M.2) — в x4. Додає швидкі диски.
6. Мережева карта 10 Гбіт — в x4 або x8. Для швидкого інтернету вдома.
7. RAID контролер — в x4/x8. Щоб кілька дисків працювали разом.
8. Riser (подовжувач) — в x16. Переставляє відеокарту в інше місце (наприклад, вертикально в корпусі).

Все, здається, це основне. Якщо треба — можна додати фотки слотів і карт з інтернету.
