# Тема 1–2  
## Внутрішні компоненти персонального комп’ютера

---

## 1. Правила електробезпеки при роботі з внутрішніми компонентами ПК

Під час роботи з внутрішніми компонентами комп’ютера необхідно дотримуватися
правил електробезпеки, оскільки неправильні дії можуть призвести до поломки
обладнання або ураження електричним струмом.

### Основні правила електробезпеки:
- комп’ютер потрібно **повністю вимкнути** та **від’єднати від електромережі**;
- після відключення рекомендується натиснути кнопку ввімкнення для зняття
  залишкової напруги;
- не торкатися контактів мікросхем і роз’ємів руками;
- уникати статичної електрики під час роботи з компонентами;
- працювати тільки в сухому приміщенні;
- не підключати блок живлення під час виконання робіт.

---

## 2. Форм-фактори корпусів настільних комп’ютерів

Форм-фактор корпусу визначає його розміри, розташування елементів та
сумісність з материнськими платами.

### Основні форм-фактори корпусів:
- **Full Tower** — великі корпуси для потужних комп’ютерів, мають багато місця
  для компонентів;
- **Mid Tower** — найпоширеніший тип корпусу, підходить для більшості ПК;
- **Mini Tower** — компактні корпуси з обмеженими можливостями розширення;
- **SFF (Small Form Factor)** — корпуси малого розміру для компактних систем.

---

## 3. Блок живлення: основні характеристики та конектори

Блок живлення призначений для забезпечення всіх компонентів комп’ютера
електричною енергією та перетворення напруги до необхідних значень.

### Основні характеристики блоку живлення:
- **потужність** (вимірюється у ватах);
- **ефективність роботи** (сертифікація 80 PLUS);
- **форм-фактор** (ATX, SFX);
- **наявність захистів** від перевантаження та короткого замикання.

### Основні конектори блоку живлення:
- **24-pin ATX** — живлення материнської плати;
- **4-pin / 8-pin CPU** — живлення процесора;
- **PCI Express 6/8-pin** — живлення відеокарти;
- **SATA Power** — підключення SSD та HDD;
- **Molex** — використовується для застарілих пристроїв.

---

## 4. Основні компоненти материнської плати

Материнська плата є основною платою комп’ютера, на якій розміщуються та
з’єднуються всі інші компоненти системи.

### Основні компоненти материнської плати (7 шт.):
1. **Сокет процесора** — призначений для встановлення центрального процесора;
2. **Чипсет** — керує взаємодією між компонентами;
3. **Слоти оперативної пам’яті (RAM)** — для встановлення модулів пам’яті;
4. **Слоти PCI Express** — для підключення відеокарти та плат розширення;
5. **Роз’єми живлення** — забезпечують подачу електроенергії на плату;
6. **BIOS / UEFI** — базова система налаштування комп’ютера;
7. **Порти введення та виведення** — USB, аудіо, мережеві та відеовиходи.
